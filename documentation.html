Welcome to PlanIT.

PlanIT is a advanced Task manager web app with support of of course, Task manager with Priority manager, Reminder manager and a clock screensaver for all your planning needs.
This documentation lets you know how exactly you can use PlanIT to maximize your productivity.

Using the Task Manager:

The task manager of PlanIT is filled with sophisticated features such as priority setting, task reorganization etc.

Here's how you can Navigate to the Tasks page:

  1. Click on the icon (if in mobile) or the menu button (if in desktop) in the nav bar to access the dropdown menu.
  2. Click on the tasks option from the dropdown menu.
  3. You will be greeted with the task manager page of PlanIT.

Adding tasks:

To add tasks, here the the following steps you should be following:
  1. Type your tasks in the add task feild box.
  2. Click on "ADD" button adjacent to the task feild or click enter key (if on desktop) to add the task
  3. Now you can see the tasks which you just added in the "Current Tasks" section.

Setting priority:

To set Priority to your tasks, Here are the following steps you should be following:
  1. Click on the select Priority Dropdown menu.
  2. Choose your priotity now.
    • Low Priority for tasks with no immediate attention.
    • Medium Priority for tasks with little immediate attention.
    • High Priority for tasks with immediate attention.

Removing the done task.

To remove the done task, Here are the following steps you should following:
  1. Click on the checkbox on the side of the task description
  2. You will now see the task in the completed tasks section with the task description strikethroughed.

By following these steps we can easily maintain the tasks in the PlanIT task manager.

Using the Reminder Manager.

The reminder manager of PlanIT is filled with features to keep you on track.

Here's how you can Navigate to the Reminders page:

  1. Click on the icon (if in mobile) or the menu button (if in desktop) in the nav bar to access the dropdown menu.
  2. Click on the reminders option from the dropdown menu.
  3. You will be greeted with the reminder manager page of PlanIT.

Adding reminders:

To add reminders, here are the following steps you should be following:
  1. Type your reminder in the add reminder field box.
  2. Click on "ADD" button adjacent to the reminder field or click enter key (if on desktop) to add the reminder
  3. Now you can see the reminders which you just added in the "Current Reminders" section.

Marking reminders as complete:

To mark a reminder as complete, Here are the following steps you should following:
  1. Click on the checkbox on the side of the reminder description.
  2. You will now see the reminder marked as complete.

By following these steps, we can easily manage reminders in the PlanIT reminder manager.

Navigating to the Clock screensaver:

The Clock screensaver is a screensaver which shows you the current time in real time, keeping you updated to the exact moment of the day.

To navigate to clock, here are the following steps you should be following:

  1. Click on the icon (if in mobile) or the menu button (if in desktop) in the nav bar to access the dropdown menu.
  2. Click on the clock option from the dropdown.
  3. This will land you to the clock screensaver page.
